== Description ==
Permite subir un zip con archivos de Leaflet (generados en Qgis), generando un link que se puede insertar o embeber en una entrada. 

== Installation ==
Instalar el plugin.
Subir Zip en la seccion "Subir Mapa Leaflet".
Buscar el link generado en la seccion "Mapas Subidos".

== Changelog ==
